# online_sales_analysis
This is my Final Project - Rad sa GitHubom i Pythonom

Projekat ima za cilj napređenje sistema za analizu prodajnih podataka u online prodavnici.
Cilj je implementacija sistema koji podržava:
- rad sa proizvodima (dodavanje, prikaz, ažuriranje i uklanjanje);
- upravljanje korpom kupca (dodavanje proizvoda, prikaz sadržaja i izračunavanje ukupne vrednosti).

Sadrži klase Product, ProductManager i Cart sa pripadajućim funkcionalnostima prikaya, dodavanja, ažuriranje, izračuna ukupne vrijednosti i uklanjanja.
